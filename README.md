# Student Performance Analysis

## Project Overview
This project analyzes student performance using Exploratory Data Analysis (EDA).

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Students who completed test preparation courses scored higher.
- Higher parental education levels are linked to better student performance.

## Project Structure
- data/ : dataset
- notebooks : EDA and analysis files
